# 期末專題

## 桌面型手部辨識滑鼠

## 組員

> S0954001 龔品宇 \
S0954005 林昀佑 \
S0954022 林煜宸 \
S0954040 吳季旻 \
S0xxxxxx 許家碩

## 流程

1. 透過影像輸入設備取得桌面手部影像 \
分辨出手部與桌面
2. 找出手部之後 \
將手部位置定為原點
3. 將目前偵測到的手部做骨架應用
4. 分別將骨架的各模式偵測出來 \
並根據各個功能去做操作
    * 左右移動 (對應滑鼠X軸)
    * 前後移動 (對應滑鼠Y軸)
    * 左鍵press
    * 左鍵release
    * 右鍵press
    * 右鍵release
    * 側鍵press
    * 側鍵release
